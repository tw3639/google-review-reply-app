# Review Reply QA

AIによるGoogleレビュー返信支援ツール  
— **「AIっぽさ」を抑えつつ、最終判断は人が行う設計** —

## 概要
Googleレビューに対する返信文を、★評価ごとの目的・トーンに応じて生成する支援ツールです。  
定型的なAI返信を避け、「人が書いたように見える自然な表現」を重視しています。

自動生成された文章をそのまま使うことを前提とせず、  
**確認・微調整・判断は人が行う**ことを前提に設計されています。

---

## 特徴
- ★1〜★5で返信の目的・役割を明確に分離
- 定型文回避を前提としたプロンプト設計
- 生成文を編集しながら確認できるUI
- **AIっぽさスコアによる参考チェック**
- CSVによる一括生成（軽量・検証向け）

---

## 主な機能
- 感情ケア・事実整理・ポジティブ返信の自動切替
- レビュー文内容に応じた簡易パーソナリティ反映
- 生成後の文章を手動で編集可能
- 編集後の文章に対してもAIっぽさスコアを再計算

---

## AIっぽさスコアについて
本ツールにおける「AIっぽさスコア」は、  
**文章の品質や正解・不正解を判定する指標ではありません。**

以下のような観点をもとに、  
「定型的・機械的に見えやすい表現の傾向」を参考値として数値化しています。

- 定型的なAI表現が多く含まれていないか
- 構文や語尾が単調になりすぎていないか
- レビュー固有の内容に十分触れているか

高スコアでも実務上まったく問題なく使える文章は多く存在します。  
あくまで  
**「自分で手直しした結果、AI感がどう変化したかを確認するための補助指標」**  
としてご利用ください。

社内検証・トーン確認などの用途を想定した軽量モードです。
