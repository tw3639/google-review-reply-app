# Review Reply QA

AIによるGoogleレビュー返信支援ツール  
— 「AIっぽさ」を抑えた自然な返信生成 —

## 特徴
- ★1〜★5で返信目的を明確に分離
- 定型文回避を前提としたプロンプト設計
- AIっぽさスコアによる自己チェック（品質評価ではありません）
- CSV一括生成（軽量）

## 主な機能
- 感情ケア・実務対応・ポジティブ返信の自動切替
- レビュー文から簡易パーソナリティ推定
- 生成後に自由編集可能、AIっぽさスコアを再計算
- 名前（〇〇様）は必要に応じて追加可能
  - 名前を入れると本人が書いた感が増す
  - 謝罪文の冒頭に限定される場合があります

## 技術構成
- Python / Streamlit
- OpenAI API
- 軽量CSV処理

## 使用上の注意
- AIっぽさスコアはあくまで補助指標です
  - 高スコア＝悪い文章、低スコア＝正解、ではありません
  - 自分で手直しした結果、定型AI感がどう変化したか確認する目的です
- 自動生成された文章は、必ず内容を確認してご利用ください

※ 社内検証・業務効率化を目的とした試作
